# nano

Text editor.

- Homepage: <https://nano-editor.org>
- Doc: <https://nano-editor.org/docs.php>
- Repo: <https://git.savannah.gnu.org/cgit/nano.git>

## Installation

- Debian package: `nano`

## Usage

```text
ctrl+k # Delete line
ctrl+6 # Select text
alt+6  # Copy selected text
ctrl+k # Cut selected text
ctrl+u # Past text
alt+u  # Undo
at+e   # Redo
ctrl+c # Save
ctr+x  # Exit
```
